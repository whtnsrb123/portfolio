# 👋 조순규 포트폴리오

### 🎮 Unity 클라이언트 개발자 (게임 & 모바일)

**도전과 성과로 성장한 소프트웨어 개발자**

어려운 문제에 꾸준히 **도전하고** 성과를 통해 **성장**해 온 소프트웨어 개발자입니다.

학창 시절부터 전국 **알고리즘 대회**에서 우수한 성적을 거두며 기초 역량을 탄탄히 다졌고, 이를 기반으로 국민대학교 **특기자 전형**으로 입학하였습니다.
학부 과정에서는 **실무형 프로젝트**와 **팀 협업 경험**을 통해 **문제 해결 능력**을,
**코딩 강사**로 활동하며 다양한 연령층 학생들을 가르치면서 복잡한 개념을 쉽게 전달하는 능력과 개별 피드백을 통한 맞춤형 지도 경험을 통해 **커뮤니케이션 역량**을 한층 키웠습니다.
또한 Unity, Photon PUN, Android Studio와 AI 도구를 적극 활용하여 기능 최적화와 네트워크 문제 해결까지 경험하며 성장의 폭을 넓혔습니다.

이처럼 기술적 도전과 협업을 통해 끊임없이 발전하는 개발자로서 혁신과 효율성 향상에 기여하고 싶습니다.
